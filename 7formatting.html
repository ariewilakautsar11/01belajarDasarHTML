Formatting

  • Dalam aplikasi pengolahan text seperti Microsoft Word, kita bisa mengubah format tulisan menjadi tebal atau miring misal
  • Begitu juga di HTML, kita bisa menambah formating di text yang ingin kita tampilkan menggunakan tag sesuai dengan format yang kita mau

    • Formatting Tag

  • <b >, <strong> untuk bold / tebal Ini contoh bold b dan Ini contoh Strong
  • <i>, <em> untuk italic / miring contoh: Ini Italic dan Ini em
  • <small> untuk text kecil Ini Contoh tag small
  • <del> untuk text dicoret Ini contoh tag del
  • <ins> untuk text dengan garis bawah Ini contoh tag ins
  • <sub> untuk subscript text Ini contoh tag sub
  • <sup> untuk superscript text Ini contoh tag sup
